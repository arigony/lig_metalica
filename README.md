# Ligação Metálica — Guia de Estudo Interativo

Página HTML única, responsiva e pronta para GitHub Pages, construída como guia de estudo educativo sobre **ligação metálica**.

## Conteúdo incluído

- Conceito de ligação metálica.
- Elétrons de valência em metais e formação de cátions.
- Modelo do “mar de elétrons”.
- Teoria dos elétrons livres.
- Rede cristalina e célula unitária.
- Estruturas CCC, CFC e HC com exemplos.
- Propriedades dos sólidos metálicos:
  - brilho;
  - maleabilidade e ductilidade;
  - ligas metálicas;
  - condutividade elétrica;
  - condutividade térmica;
  - resistividade e temperatura;
  - altos pontos de fusão e ebulição.
- Tabelas de condutividade, ligas e pontos de fusão/ebulição.
- Flashcards e quiz com feedback formativo.
- Controle manual para iniciar/pausar animações, evitando que a página rode sozinha e reduzindo distração/carga no celular.

## Simulações interativas

1. Mar de elétrons com controle de elétrons de valência e temperatura; as animações começam pausadas e só iniciam quando o estudante clicar em **Iniciar animações**.
2. Células unitárias CCC, CFC e HC.
3. Brilho metálico por oscilação eletrônica.
4. Maleabilidade e ductilidade por deslocamento de cátions e redistribuição eletrônica.
5. Condutividade elétrica/térmica com campo elétrico e temperatura.
6. Formação de ligas como soluções sólidas.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie o arquivo `index.html` e este `README.md` para a raiz do repositório.
3. Abra **Settings > Pages**.
4. Em **Build and deployment**, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salve e aguarde o link do GitHub Pages.

## Observação didática

As animações são modelos simplificados para fins de ensino em Química Geral. Elas não substituem modelos quânticos completos de estrutura eletrônica/bandas, mas ajudam o estudante a conectar o modelo microscópico às propriedades macroscópicas dos metais.
